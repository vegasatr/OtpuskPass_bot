# Git Scripts

Папка содержит скрипты для автоматизации работы с Git.

## Файлы

### git_push.bat (Windows)
Скрипт для автоматического пуша проекта на GitHub с увеличением версии.

**Использование:**
```batch
git\git_push.bat
```

Или с описанием изменений:
```batch
git\git_push.bat "Добавлена новая функция"
```

**Что делает скрипт:**
1. Автоматически увеличивает версию проекта (patch версию)
2. Обновляет файл `version.txt`
3. Анализирует измененные файлы
4. Генерирует умное описание изменений (или использует ваше)
5. Создает коммит
6. Создает новую ветку с именем версии
7. Отправляет изменения на GitHub
8. Создает тег релиза

### git_push.sh (Linux/Mac)
Аналогичный скрипт для Linux и Mac OS.

**Использование:**
```bash
bash git/git_push.sh
```

Или с описанием изменений:
```bash
bash git/git_push.sh "Добавлена новая функция"
```

## Примеры

### Автоматическое описание
```batch
git\git_push.bat
```
Скрипт сам проанализирует изменения и создаст описание.

### Своё описание
```batch
git\git_push.bat "Исправлена ошибка в обработке платежей"
```

## Требования

- Git должен быть установлен и доступен из командной строки
- Репозиторий должен быть инициализирован
- Должен быть настроен удаленный репозиторий `origin`
- Файл `version.txt` должен существовать в корне проекта

## Формат версии

Версия должна быть в формате `x.y.z` (например, `1.0.7`).
Скрипт автоматически увеличивает последнюю цифру (patch версию).
